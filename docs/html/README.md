# HTML

## noopener, noreferrer
```
<a rel="noopener noreferrer">
```
* `noopener` location 변경과 같은 자바스크립트요청을 거부시킨다.
* `noreferrer` href 에 설정한 링크에서 클릭한 사용자의 정보를 알 수 없도록 함
